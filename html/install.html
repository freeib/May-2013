Installation instructions

This CD-ROM includes HTML (hypertext markup language) files for the navigation pages and PDF (portable document format) files for the examination papers, markschemes and list of contents. For the music listening paper only, MP3 files are provided.

If you have a web browser to read HTML pages (for example, Internet Explorer®, Firefox®, Safari® or Google Chrome™), Adobe® Reader® to read PDF files, and MP3 player software to listen to music files, you are ready to access the contents of the CD-ROM.

You can also open the PDF files directly by using Windows Explorer (Windows®), Finder (Mac OS® X) or other file browser (Linux®) to navigate the CD-ROM.

To install Adobe Reader

The most recent version of Adobe Reader can be downloaded from the internet.

To install an MP3 player

Windows

Your computer may already have Windows Media® Player installed, in which case clicking on one of the musical extract links in group 6 will play the track. If not, you can use the following links to download Windows Media Player or iTunes®.

Mac OS X

Your computer may already have iTunes installed, in which case clicking on one of the musical extract links in group 6 will play the track. If not, you can use the following link to download iTunes®.

Linux

Your computer may already have an MP3 player installed, in which case clicking on one of the musical extract links in group 6 will play the track. If not, there are a number of MP3 players for Linux; consult the documentation for your particular distribution.

You can now open the file "index.html" in your web browser to start navigating the CD-ROM.

Usage tip: When you click on a link to an examination paper or markscheme, the PDF file may open in the browser window or a new window, depending on the settings of your computer. You can use the browser back button to return to the navigation pages. To ensure PDF files open in a new window, right-click on the link (right-handed mouse users) and select "Open in New Window" (Internet Explorer) or "Open Link in New Window" (Firefox).
